# customer_shopping_behavior
End-to-end retail analytics project analysing customer shopping behaviour, purchase trends, and product preferences to answer business questions and build an interactive sales dashboard.

•	Performed Exploratory Data Analysis (EDA) using Python and Pandas — data import, exploration, cleaning, type fixing, and identifying key customer and purchase patterns

•	Loaded the cleaned dataset into PostgreSQL using SQLAlchemy and answered 15+ business SQL queries covering sales trends, customer demographics, purchase frequency, product preferences, and revenue drivers

•	Designed an interactive Power BI dashboard connected to the SQL database — visualised customer segments, top-selling categories, seasonal trends, and revenue KPIs with dynamic slicers

•	Delivered a structured project report and presentation deck summarising methodology, insights, and business recommendations for stakeholder review
